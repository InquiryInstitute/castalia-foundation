# Castalia Foundation

Castalia Foundation is a 501(c)(3) non-profit. This repo hosts the foundation site at [castalia.foundation](https://castalia.foundation).

- **Site:** https://castalia.foundation  
- **Institute:** https://castalia.institute  

## GitHub Pages

- Deploy from branch `main`, root.
- Custom domain: `castalia.foundation` (set in repo Settings → Pages).
- DNS: CNAME `castalia.foundation` → `inquiryinstitute.github.io` (Route 53).
